웹사이트란?

URL(Uniform Resource Locator) 등을 통하여 보이는
웹 페이지의 묶음

웹사이트를 구성하는 언어

HTML(HyperText Mark-up Language)

CSS(Cascading Style Sheet)

JavaScript(HyperText Mark-up Language)

HTML이란?

HYPERTEXT
사용자에게 비순차적인 검색을 할 수 있도록 제공되는 텍스트MARK-UP
문서 처리의 원활한 지원을 위해 문서에 추가되는 약속된 정보LANGUAGE
언어

= 웹사이트에 다양한 정보와 이를 찾기 위한 하이퍼텍스트를
     '마크업'하여 사용자가 볼 수 있도록 하는 언어

HTML의 기본 문법

<p class="value">content</p>

<p 
시작 태그class
속성 이름="value
속성 값">content
내용</p
종료 태그>

div 태그

Division(분할)의 약자로, 화면의 레이아웃을 나눌 때 쓰임

<div></div>

p 태그

Paragraph(단락)의 약자로, 텍스트를 삽입할 때 쓰임

<p>텍스트</p>

img 태그

Image(사진)의 약자로, 사진을 삽입할 때 쓰임

<img src="(사진 링크)">

a 태그

Anchor(닻)의 약자로, 하이퍼링크를 삽입할 때 쓰임

<a href="(하이퍼링크)">(요소)</a>

CSS란?

CASCADING
폭포수처럼 떨어지는STYLE SHEET
글꼴·레이아웃 설정으로 구성된 파일

= 폭포수가 떨어지는 모습처럼 종속적으로
     연결되어 HTML 요소를 꾸며주는 언어

div {width: 100px;}
div > div {width: 50px;}
div > div > p {font-size: 20px;}
div > div > img {height: 10px;}
div > div > span {font-size: 10px;}
div > div > span > strong {font-weight: bold;}

CSS의 기본 문법

div {background: red; width: 50px;}

background

요소의 배경에 색, 사진 등을 넣을 때 쓰임

div {background: red;}

color

텍스트의 색을 설정할 때 쓰임

p {color: red;}

font-size

텍스트의 크기를 설정할 때 쓰임

p {font-size: 60px;}

width, height

요소의 크기를 설정할 때 쓰임

div {width: 100px;}
